# 🐱 Katzen-Website – Russisch Blau

> **🌐 Live Demo: [katze-russisch-blau.netlify.app](https://katze-russisch-blau.netlify.app/)**

Eine Informationswebsite über die Katzenrasse **Russisch Blau** – mit Steckbrief, Charakterbeschreibung und Bildern.

---

## 📋 Über das Projekt

Diese Website wurde als Lernprojekt im Bereich Webentwicklung erstellt. Sie stellt die Katzenrasse Russisch Blau vor und enthält:

- Allgemeine Informationen zur Rasse
- Detaillierter Steckbrief (Größe, Gewicht, Charakter, Fellpflege u. v. m.)
- Bildergalerie mit Beispielfotos der Rasse

## 🐾 Über die Russisch Blau

Die Russisch Blau ist eine besonders elegante und sensible Katzenrasse. Sie ist bekannt für ihr charakteristisches blaugraues Fell mit Silberschimmer und ihre leuchtend grünen Augen.

| Merkmal | Details |
|---|---|
| Größe | Klein bis mittelgroß |
| Gewicht | 2,5 – 6,5 kg |
| Lebenserwartung | Bis zu 20 Jahre |
| Charakter | Ruhig, intelligent, anhänglich |
| Besonderheit | Sehr sensibel – keine Anfängerkatze |
